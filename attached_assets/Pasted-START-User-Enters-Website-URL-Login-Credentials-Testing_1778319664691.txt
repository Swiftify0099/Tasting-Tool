START
  ↓
User Enters:
- Website URL
- Login Credentials
- Testing Goal

  ↓
Open Browser (Playwright)

  ↓
Go To URL

  ↓
Extract DOM
(inputs, buttons, links, forms)

  ↓
Compress DOM
(remove useless HTML/CSS/scripts)

  ↓
Send To AI (Qwen)

  ↓
AI Analyze Page
- detect forms
- detect buttons
- detect validations
- detect navigation
- detect possible flows

  ↓
AI Returns JSON Actions

Example:
{
 "tool":"type",
 "selector":"#email",
 "text":"test@gmail.com"
}

  ↓
Playwright Executes Action

  ↓
Take Screenshot

  ↓
Wait For:
- page load
- network idle
- route change
- popup

  ↓
Extract NEW DOM

  ↓
Compare URL

IF URL CHANGED:
   ↓
   Add New Route To Queue

Example:
Queue:
- /dashboard
- /profile
- /settings

   ↓
Mark Previous Page As Tested

  ↓
Discover More Routes

AI + Backend finds:
- sidebar links
- navbar links
- buttons
- hidden forms

  ↓
Add Untested Routes To Queue

  ↓
Go Next Route From Queue

  ↓
Repeat Process

Take Screenshot

- alternative selector

  ↓
Retry Test
